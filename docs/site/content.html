Content Module

Content loading pipeline, JSON content packs, map definitions, world generation, and mod system. Build games with data-driven content that can be extended without code changes.

ContentPack

A JSON bundle containing items, creatures, recipes, and terrain definitions. Content packs are the primary way to add game content.

interface ContentPack {
    id: string;                    // Unique pack identifier
    name: string;                  // Display name
    version: string;               // Semantic version
    author?: string;               // Creator name
    description?: string;          // Description
    dependencies?: string[];       // Required pack IDs
    items?: ItemTemplate[];        // Item definitions
    creatures?: CreatureTemplate[]; // Creature definitions
    recipes?: Recipe[];            // Crafting recipes
    terrains?: TerrainDefinition[]; // Custom terrain
}

CreatureTemplate

Template for spawning creature entities.

interface CreatureTemplate {
    id: string;
    name: string;
    description: string;
    character: string;             // Display char
    foreground: string;            // Color
    background?: string;
    health: number;
    speed: number;                 // Action speed
    faction?: string;              // 'friendly', 'hostile', etc.
    tags: string[];
    ai?: {
        type: string;              // AI behavior type
        aggression: number;        // 0-1 aggression level
        visionRange: number;       // Sight distance
    };
    loot?: string[];               // Item template IDs to drop
}

Recipe

Crafting recipe definition.

interface Recipe {
    id: string;
    name: string;
    description: string;
    category: 'crafting' | 'construction' | 'cooking';
    ingredients: RecipeIngredient[];
    results: RecipeIngredient[];
    timeCost: number;              // Ticks to craft
    requiredTools?: string[];      // Required tool item IDs
    requiredSkills?: Record;  // Skill requirements
    station?: string;              // Required crafting station
}

interface RecipeIngredient {
    itemId: string;
    quantity: number;
}

MapDefinition

Defines a complete game world with size, generator, and spawn points.

interface MapDefinition {
    id: string;
    name: string;
    description: string;
    width: number;                 // World width in tiles
    height: number;                // World height in tiles
    chunkSize: number;             // Chunk dimensions (usually 64)
    generator?: string;            // Generator name: 'wilderness', 'dungeon', 'cave', 'city'
    generatorParams?: Record;  // Generator-specific params
    predefinedChunks?: PredefinedChunk[];       // Hand-crafted chunks
    spawnPoints?: SpawnPoint[];    // Player/NPC spawn locations
}

MapLoader

Loads map definitions and creates game worlds.

new MapLoader()
constructor(contentLoader: ContentLoader, eventBus: EventBus)
const mapLoader = new MapLoader(contentLoader, eventBus);
loadMapDefinition()
loadMapDefinition(definition: MapDefinition): void
mapLoader.loadMapDefinition({
    id: 'dungeon_level_1',
    name: 'Level 1 - The Entrance',
    description: 'The upper level of the ancient dungeon',
    width: 500,
    height: 500,
    chunkSize: 64,
    generator: 'dungeon',
    generatorParams: { roomChance: 0.4, seed: 12345 },
    spawnPoints: [
        { x: 32, y: 32, type: 'player', tags: ['start'] }

WorldGenerator

Pluggable world generation system with built-in generators.

registerGenerator()
registerGenerator(name: string, generator: ChunkGenerator): void
Register a custom chunk generator function.
mapLoader.worldGenerator.registerGenerator('maze', (chunk, context) => {
    // Generate maze-like terrain in this chunk
    for (let y = 0; y < chunk.size; y++) {
        for (let x = 0; x < chunk.size; x++) {
            // Your generation logic here
            chunk.setTile(x, y, TERRAIN.floor);
        }
    }
});
generateWorld()
generateWorld(world: World, generatorName: string, params?: Record<string, unknown>): void
Apply a generator to a world. Generator runs lazily as chunks are accessed.
mapLoader.worldGenerator.generateWorld(world, 'cave', {
    fillPercent: 0.45,
    iterations: 5,
    seed: 12345
});

Built-in Generators

GeneratorDescriptionParameters
wilderness Forests, clearings, and water treeDensity (0-1), waterChance (0-1)
dungeon Room-based dungeons with corridors roomChance (0-1)
cave Cellular automata caves fillPercent (0-1), iterations
city Grid streets and buildings blockSize, buildingDensity (0-1)

GeneratorContext

Passed to generator functions.

interface GeneratorContext {
    world: World;                  // The world being generated
    rng: () => number;            // Seeded random number generator (0-1)
    params: Record;  // Generator parameters
}

type ChunkGenerator = (chunk: Chunk, context: GeneratorContext) => void;

Mod System

JavaScript-based modding with full API access.

Mod Interface

interface Mod {
    id: string;
    name: string;
    version: string;
    author?: string;
    description?: string;
    dependencies?: string[];
    initialize(api: ModAPI): void;  // Called when mod loads
    cleanup?(): void;               // Optional cleanup
}

ModAPI

API provided to mods for content registration.

interface ModAPI {
    eventBus: EventBus;
    contentLoader: ContentLoader;
    mapLoader: MapLoader;
    worldGenerator: WorldGenerator;
    registerItem: (template: ItemTemplate) => void;
    registerCreature: (template: CreatureTemplate) => void;
    registerRecipe: (recipe: Recipe) => void;
    registerTerrain: (definition: TerrainDefinition) => void;
    registerGenerator: (name: string, generator: ChunkGenerator) => void;
    createEntity: (templateId: string, x: number, y: number, ecsWorld: ECSWorld) => Entity | null;
    spawnItem: (templateId: string, quantity: number, x: number, y: number, itemManager: any) => any;
}

Complete Example

import { ContentManager } from 'ascii-world';

// Create content manager
const content = new ContentManager(eventBus);

// Load content pack from JSON
const packJson = await fetch('content/fantasy_pack.json').then(r => r.text());
content.loadContentPack(packJson);

// Load map definition
const mapJson = await fetch('maps/dungeon.json').then(r => r.text());
content.loadMap(mapJson);

// Load a JavaScript mod
const combatMod: Mod = {
    id: 'enhanced_combat',
    name: 'Enhanced Combat',
    version: '1.0.0',
    initialize(api) {
        // Add new weapon
        api.registerItem({
            id: 'sword_great',
            name: 'Greatsword',
            category: ItemCategory.WEAPON,
            character: '/',
            foreground: '#silver',
            properties: {
                weight: 4,
                volume: 5,
                durability: 200,
                value: 100
            }
        });

        // Listen to events
        api.eventBus.on('combat:hit', (data) => {
            console.log(`Combat hit: ${data.damage} damage`);
        });
    }
};

content.loadMod(combatMod);
content.initializeMods(itemManager);

// Create world
const world = content.createWorld('dungeon_level_1', ecsWorld);
console.log('World ready!');
