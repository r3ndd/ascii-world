Actor Interface

Actors participate in the turn system. Implement this interface for any entity that takes turns. 

interface Actor {
    entityId: EntityId;
    getSpeed(): number;
    act(): Promise<void> | void;
}

Example Actor Implementation

class EntityActor implements Actor {
    entityId: EntityId;
    private entity: Entity;
    private ai: AISystem;

    constructor(entity: Entity, ai: AISystem) {
        this.entity = entity;
        this.entityId = entity.id;
        this.ai = ai;
    }

    getSpeed(): number {
        const speed = this.entity.getComponent<SpeedComponent>('speed');
        return speed?.value || 100;
    }

    async act(): Promise<void> {
        await this.ai.takeTurn(this.entity);
    }
}

Speed System

Calculates action costs based on actor speed. Speed 100 is normal, higher is faster, lower is slower.

Speed Action Cost Multiplier Description
200 0.5x Fast (acts twice as often)
100 1.0x Normal
50 2.0x Slow (acts half as often)
calculateActionCost()
calculateActionCost(baseCost: number, speed: number): number 
const cost = speedSystem.calculateActionCost(100, 150);
// Returns 67 (100 * 100 / 150)

Deferred Updates

Handle "catch-up" for entities in inactive chunks when they re-enter the active zone.

DeferredUpdateSystem

const deferredSystem = new DeferredUpdateSystem(ecsWorld, eventBus);

// Queue an entity for catch-up
deferredSystem.queueCatchUp(entityId, missedTurns);

// Process all queued updates
deferredSystem.processAll();

CatchUpCalculator

Calculates accumulated effects over missed turns (hunger, regeneration, etc.). 

const results = CatchUpCalculator.calculate(entity, 50, 100);
console.log(results.healthDelta);   // Health regenerated
console.log(results.hungerDelta);   // Hunger accumulated
console.log(results.events);        // Summary events

Complete Example

import {
    TurnManager,
    SpeedSystem,
    Action,
    globalEventBus
} from 'ascii-world';

// Create turn manager
const turnManager = new TurnManager(ecsWorld, globalEventBus, new SpeedSystem());

// Create player actor
class PlayerActor implements Actor {
    entityId = player.id;

    getSpeed(): number {
        const speed = player.getComponent<SpeedComponent>('speed');
        return speed?.value || 100;
    }

    async act(): Promise<void> {
        // Player turn - wait for input
        await new Promise(resolve => {
            document.addEventListener('keydown', function handler(e) {
                if (handleInput(e)) {
                    document.removeEventListener('keydown', handler);
                    resolve(undefined);
                }
            });
        });
    }
}

// Register player
const playerActor = new PlayerActor();
turnManager.registerActor(playerActor, true);

// Start turn loop
turnManager.start();
Tip: Use the SpeedSystem to create varied gameplay. Fast enemies feel frantic, while slow enemies give players time to react.
