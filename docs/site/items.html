Items Module

Item system with templates, inventory management, stacking, equipment, and durability. Supports weight/volume constraints and serialization for save/load.

ItemCategory

Enumeration of item categories for organization and filtering.

Category Description
WEAPON Melee and ranged weapons
ARMOR Protective equipment
CONSUMABLE Food, potions, medical items
TOOL Crafting and utility tools
MATERIAL Raw crafting materials
CONTAINER Bags, boxes, storage
MISC Miscellaneous items

ItemTemplate

Blueprint for creating item instances. Templates define the base properties of an item type. 

interface ItemTemplate {
    id: string;                    // Unique template identifier
    name: string;                  // Display name
    description: string;           // Flavor text
    category: ItemCategory;        // Item category
    character: string;             // ASCII display character
    foreground: string;            // Text color (hex)
    background?: string;           // Background color (hex)
    properties: ItemProperties;    // Physical properties
    tags?: string[];               // Optional tags for filtering
}

ItemProperties

Physical and gameplay properties of an item.

Property Type Description
weight number Weight in kilograms
volume number Volume in liters
durability number? Current durability (optional)
maxDurability number? Maximum durability (optional)
stackable boolean? Can stack with identical items
maxStack number? Maximum stack size
value number? Base monetary value
quality number? Quality level 1-100
metadata object? Custom properties

Item

Represents an item instance in the game world. Items can exist on the ground, in inventories, or be equipped.

id
get id(): string
Unique instance identifier. 
console.log(item.id);  // "item_1"
name
get name(): string
console.log(item.name);  // "Iron Sword"
quantity
get quantity(): number
set quantity(value: number)
Number of items in this stack. Emits item:quantityChanged event on change. 
item.quantity += 5;  // Add 5 to stack
weight
get weight(): number
Total weight including quantity (weight * quantity). 
console.log(item.weight);  // 3.0 kg for 2 swords at 1.5kg each
volume
get volume(): number
Total volume including quantity (volume * quantity).
damage()
damage(amount: number): void
Apply damage to item durability. Emits item:damaged or item:broken events. 
item.damage(10);  // Reduce durability by 10
if (item.durability <= 0) {
    console.log('Item broke!');
}
repair()
repair(amount: number): void
Repair item durability. Cannot exceed maxDurability. Emits item:repaired event. 
item.repair(25);  // Restore 25 durability
isStackable()
isStackable(): boolean 
if (item.isStackable()) {
    // Can stack with other items
}
isEquipped()
isEquipped(): boolean 
if (item.isEquipped()) {
    console.log(`${item.name} is equipped`);
}
setEquipped()
setEquipped(equipped: boolean): void
Set equipped status. Emits item:equippedChanged event. 
item.setEquipped(true);  // Equip item
canStackWith()
canStackWith(other: Item): boolean
Check if this item can stack with another (same template, quality). 
if (itemA.canStackWith(itemB)) {
    itemA.quantity += itemB.quantity;
}

ItemManager

Central registry for item templates and active item instances. Handles item spawning and lifecycle.

// Auto-registers default templates (sword_iron, potion_health, etc.)

Register a new item template. Emits item:templateRegistered event. 
itemManager.registerTemplate({
    id: 'excalibur',
    name: 'Excalibur',
    description: 'Legendary sword of the king',
    category: ItemCategory.WEAPON,
    character: '/',
    foreground: '#ffd700',
    properties: {
        weight: 2.0,
        volume: 3,
        durability: 200,
        maxDurability: 200,
        value: 10000
    }
});

spawnItem(templateId: string, quantity?: number, location?: Position): Item | null
Create a new item instance from a template. Emits item:spawned event. 
// Spawn a sword on the ground at position
const sword = itemManager.spawnItem('sword_iron', 1, { x: 100, y: 200 });

// Spawn a stack of potions
const potions = itemManager.spawnItem('potion_health', 5);
getItem()
getItem(id: string): Item | undefined

removeItem(id: string): boolean
Remove an item instance. Emits item:removed event. 
itemManager.removeItem('item_1');

getItemsAt(location: Position): Item[]

Complete Example

import { ItemManager, InventoryManager, ItemCategory } from 'ascii-world';

// Create managers
const itemManager = new ItemManager(eventBus);
const inventoryManager = new InventoryManager(itemManager, eventBus);

// Register custom item template
itemManager.registerTemplate({
    id: 'magic_sword',
    name: 'Magic Sword',
    description: 'A sword glowing with magical energy',
    category: ItemCategory.WEAPON,
    character: '/',
    foreground: '#00ffff',
    properties: {
        weight: 1.5,
        volume: 3,
        durability: 100,
        maxDurability: 100,
        value: 500
    }
});

// Create player inventory
const playerInv = inventoryManager.createInventory(playerEntity.id, 50, 30);

// Spawn and add items
const sword = itemManager.spawnItem('magic_sword', 1);
if (playerInv.addItem(sword)) {
    playerInv.equipItem(sword.id);
    console.log(`Equipped ${sword.name}!`);
}

// Stackable items
const arrows = itemManager.spawnItem('arrow', 20, { x: 100, y: 200 });
if (playerInv.addItem(arrows)) {
    console.log(`Picked up ${arrows.quantity} arrows`);
}

// Check inventory status
console.log(`Weight: ${playerInv.currentWeight}/${playerInv.weightCapacity} kg`);
console.log(`Items: ${playerInv.getItemCount()}`);

// Get all weapons
const weapons = playerInv.getItemsByCategory(ItemCategory.WEAPON);
for (const weapon of weapons) {
    console.log(`Weapon: ${weapon.name} (Durability: ${weapon.durability})`);
}

Events

Event Payload Description
item:templateRegistered { templateId } New template registered
item:spawned { itemId, templateId, quantity, location? } Item instance created
item:removed { itemId } Item instance destroyed
item:quantityChanged { itemId, oldValue, newValue } Stack quantity changed
item:damaged { itemId, damage, remainingDurability } Item took damage
item:broken { itemId } Item durability reached 0
item:repaired { itemId, amount, durability } Item was repaired
item:equippedChanged { itemId, equipped } Equip status changed
inventory:itemAdded { ownerId, itemId, weight, volume } Item added to inventory
inventory:itemRemoved { ownerId, itemId, weight, volume } Item removed from inventory
inventory:stacked { ownerId, targetItemId, sourceItemId, quantity } Items stacked together
inventory:itemDropped { ownerId, itemId, location } Item dropped on ground
inventory:itemTransferred { fromOwnerId, toOwnerId, itemId } Item moved between inventories
Default Templates: ItemManager automatically registers sword_iron, potion_health, wood_plank, and backpack templates on initialization.
