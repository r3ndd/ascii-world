Items Module

Item system with ECS-based architecture. Items are first-class entities with ItemComponent and ItemTemplateComponent, enabling full integration with physics, rendering, lighting, and all ECS systems. Supports inventory management, stacking, equipment, and durability.

ItemCategory

Enumeration of item categories for organization and filtering.

Category Description
WEAPON Melee and ranged weapons
ARMOR Protective equipment
CONSUMABLE Food, potions, medical items
TOOL Crafting and utility tools
MATERIAL Raw crafting materials
CONTAINER Bags, boxes, storage
MISC Miscellaneous items

ItemTemplate

Blueprint for creating item entities. Templates define the base properties of an item type and are registered in the ItemManager. 

interface ItemTemplate {
    id: string;                    // Unique template identifier
    name: string;                  // Display name
    description: string;           // Flavor text
    category: ItemCategory;        // Item category
    character: string;             // ASCII display character
    foreground: string;            // Text color (hex)
    background?: string;           // Background color (hex)
    properties: ItemProperties;    // Physical properties
    tags?: string[];               // Optional tags for filtering
}

ItemProperties

Physical and gameplay properties of an item.

Property Type Description
weight number Weight in kilograms
volume number Volume in liters
durability number? Current durability (optional)
maxDurability number? Maximum durability (optional)
stackable boolean? Can stack with identical items
maxStack number? Maximum stack size
value number? Base monetary value
quality number? Quality level 1-100
metadata object? Custom properties

ItemComponent

Runtime state component attached to item entities. Stores instance-specific data like quantity, durability, and equipment status. 

interface ItemComponent extends Component {
    type: 'item';
    templateId: string;           // Reference to template
    quantity: number;             // Stack size
    durability?: number;          // Current durability
    maxDurability?: number;       // Max durability
    quality?: number;             // Quality 1-100
    stackable: boolean;           // Can stack
    maxStack: number;             // Max stack size
    value: number;                // Base value
    equipped: boolean;            // Currently equipped
    equippedSlot?: string;        // Equipment slot
}

Accessing Item Data

// Get item component from entity
const itemComp = entity.getComponent('item');
if (itemComp) {
    console.log(`Quantity: ${itemComp.quantity}`);
    console.log(`Durability: ${itemComp.durability}/${itemComp.maxDurability}`);
}

// Modify durability
itemComp.durability = Math.max(0, itemComp.durability - 10);

// Update quantity
itemComp.quantity += 5;

ItemTemplateComponent

Static properties component derived from the item template. Contains display and physical properties. 

interface ItemTemplateComponent extends Component {
    type: 'item_template';
    name: string;                 // Display name
    description: string;          // Flavor text
    category: ItemCategory;       // Item category
    weight: number;               // Weight in kg
    volume: number;               // Volume in liters
    tags: string[];               // Filter tags
}

Calculating Total Weight/Volume

const itemComp = entity.getComponent('item');
const templateComp = entity.getComponent('item_template');

if (itemComp && templateComp) {
    const totalWeight = templateComp.weight * itemComp.quantity;
    const totalVolume = templateComp.volume * itemComp.quantity;
    console.log(`${templateComp.name}: ${totalWeight}kg, ${totalVolume}L`);
}

ECS Queries

Predefined query patterns for finding items in the ECS world.

Query Description
ITEMS_QUERY All entities with items
EQUIPPED_ITEMS_QUERY Items currently equipped
WORLD_ITEMS_QUERY Items on the ground (have position)
INVENTORY_ITEMS_QUERY Items in inventories (no position)
STACKABLE_WORLD_ITEMS_QUERY Stackable items on ground

Query Examples

import { WORLD_ITEMS_QUERY, EQUIPPED_ITEMS_QUERY } from './items/queries';

// Find all items on the ground
const worldItems = ecsWorld.queryEntities(WORLD_ITEMS_QUERY);

// Find equipped items
const equipped = ecsWorld.queryEntities(EQUIPPED_ITEMS_QUERY);

// Custom query: Find all weapons
const weapons = ecsWorld.queryEntities({
    all: ['item', 'item_template'],
    any: []
}).filter(entity => {
    const template = entity.getComponent('item_template');
    return template?.category === ItemCategory.WEAPON;
});

Inventory

Manages items for a container entity with weight/volume constraints and automatic stacking. Works with ECS entities.

new Inventory()
constructor( ownerId: EntityId, capacity: number, volumeCapacity: number, eventBus: EventBus ) 
const inventory = new Inventory(
    playerEntity.id,
    50,     // 50kg weight limit
    30,     // 30L volume limit
    eventBus
);
addItem()
addItem(ecsWorld: ECSWorld, itemEntity: Entity): boolean
Add item entity to inventory. Automatically stacks if possible. Removes PositionComponent and RenderableComponent from entity (now in inventory). Emits inventory:itemAdded or inventory:stacked events. 
const itemEntity = itemManager.spawnItem(ecsWorld, 'sword_iron', 1);
if (inventory.addItem(ecsWorld, itemEntity)) {
    console.log('Item added to inventory');
} else {
    console.log('Inventory full!');
}
removeItem()

Remove item from inventory. Returns the entity which can then be dropped or transferred. Emits inventory:itemRemoved event.

Item Systems

ECS systems that process items automatically.

ItemStackingSystem

Automatically merges stackable items at the same position in the world. Runs each update tick. 

import { ItemStackingSystem } from './items/systems';

// Register with ECS world
ecsWorld.addSystem(new ItemStackingSystem());

ItemDurabilitySystem

Processes durability changes and handles broken items.

Complete Example

import {
    ItemManager,
    InventoryManager,
    ItemCategory,
    ItemComponent,
    ItemTemplateComponent,
    WORLD_ITEMS_QUERY
} from 'ascii-world';

// Create managers
const itemManager = new ItemManager(eventBus);
const inventoryManager = new InventoryManager(eventBus);

// Register custom item template
itemManager.registerTemplate({
    id: 'magic_sword',
    name: 'Magic Sword',
    description: 'A sword glowing with magical energy',
    category: ItemCategory.WEAPON,
    character: '/',
    foreground: '#00ffff',
    properties: {
        weight: 1.5,
        volume: 3,
        durability: 100,
        maxDurability: 100,
        value: 500
    }
});

// Create player inventory
const playerInv = inventoryManager.createInventory(playerEntity.id, 50, 30);

// Spawn and add items
const sword = itemManager.spawnItem(ecsWorld, 'magic_sword', 1);
if (sword && playerInv.addItem(ecsWorld, sword)) {
    playerInv.equipItem(ecsWorld, sword.id);

    const template = sword.getComponent('item_template');
    console.log(`Equipped ${template?.name}!`);
}

// Stackable items
const arrows = itemManager.spawnItem(ecsWorld, 'arrow', 20, { x: 100, y: 200 });
if (arrows && playerInv.addItem(ecsWorld, arrows)) {
    const itemComp = arrows.getComponent('item');
    console.log(`Picked up ${itemComp?.quantity} arrows`);
}

// Check inventory status
console.log(`Weight: ${playerInv.getCurrentWeight(ecsWorld)}/50 kg`);
console.log(`Items: ${playerInv.getItems(ecsWorld).length}`);

// Find all weapons using ECS query
const weapons = ecsWorld.queryEntities({
    all: ['item', 'item_template']
}).filter(entity => {
    const template = entity.getComponent('item_template');
    const itemComp = entity.getComponent('item');
    return template?.category === ItemCategory.WEAPON &&
           playerInv.getItems(ecsWorld).includes(entity);
});

for (const weapon of weapons) {
    const template = weapon.getComponent('item_template');
    const itemComp = weapon.getComponent('item');
    console.log(`Weapon: ${template?.name} (Durability: ${itemComp?.durability})`);
}

// Damage an item
const itemComp = sword.getComponent('item');
if (itemComp) {
    itemComp.durability -= 10;
    if (itemComp.durability <= 0) {
        console.log('Weapon broke!');
        itemManager.removeItem(ecsWorld, sword.id);
    }
}

InventoryComponent

For ECS entities that have inventories, use the InventoryComponent to store inventory data directly on the entity. 

interface InventoryComponent extends Component {
    type: 'inventory';
    capacity: number;
    volumeCapacity: number;
    itemIds: EntityId[];  // IDs of items in inventory
}

// Create component
function createInventoryComponent(
    capacity: number,
    volumeCapacity: number
): InventoryComponent {
    return {
        type: 'inventory',
        capacity,
        volumeCapacity,
        itemIds: []
    };
}

// Add to entity
playerEntity.addComponent(createInventoryComponent(50, 30));

Events

Event Payload Description
item:templateRegistered { templateId } New template registered
item:spawned { entityId, templateId, quantity } Item entity created
item:removed { entityId } Item entity destroyed
inventory:itemAdded { ownerId, itemId, templateId } Item added to inventory
inventory:itemRemoved { ownerId, itemId } Item removed from inventory
inventory:stacked { ownerId, targetItemId, sourceItemId, quantity } Items stacked together
inventory:itemDropped { ownerId, itemId, location } Item dropped on ground
inventory:itemTransferred { fromOwnerId, toOwnerId, itemId } Item moved between inventories
